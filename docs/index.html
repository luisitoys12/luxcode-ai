Instalación

Listo en 3 minutos

Instala la extensión y empieza a generar páginas ahora.

1
Descarga el archivo .vsix desde la sección de Releases en GitHub
2
Instala en VS Code — Ve a Extensions Ctrl+Shift+X → menú ··· → Install from VSIX
3
Configura tu API Key — Abre el panel ⚡ en la barra lateral e ingresa tu key de Gemini o OpenAI
4
¡Genera tu primera página! — Escribe la descripción y haz clic en 🚀 Generar con IA
